Accordion

The word "accordion" may be imagined as a musical instrument of your favorite band, but in web design/developing accordion is a collapsible component for presenting information in limited space. On start user see only headers but when he click on one of them it will expand its content like the musical instrument. It might be used with menus, widgets, contents areas, etc.

One/multi collapsing

Dropdown

Dropdown is a list of items that appears when user click on.

Simple dropdown

There are two basic classes: .dropdown--out and .dropdown--in. The first one has absolutely positioned content and second one has content inside menu so when dropdown is clicked then content below will slide down to make place. Moreover the first one can have one of four additional class: .dropdown--down .dropdown--up .dropdown--left .dropdown--right. It determines on which side of container content will appear.

Content position

Tabs

Like in accordion, this component is used to show informations in limited amount of space. Instead of accordion, tabs shows data in form of table or bookmarks. Note: Because of absolute positioning of each tab content it need to set to the .tab__content class height property (default: 200px) and equivalent margin-bottom for tabs container.
